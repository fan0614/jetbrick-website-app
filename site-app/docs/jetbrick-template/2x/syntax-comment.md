注释 Comment
===========================

行注释
--------------------------

隐藏行注释的内容，以换行符结束，用于注解过程，或屏蔽指令内容。

语法：

* `#//` ...
* `##` ...

示例：

```
${user.name} #// This is a line comment.
${user.name} ##  This is a line comment.
```

块注释
-------------------------

隐藏块注释内容，可包含换行符，用于注解过程，或屏蔽指令内容。

语法：

* `#--` ... `--#`

示例：

```
#--
	This is a block comment.
--#
```
